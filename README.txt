##	To reproduce all the tables in the paper, appendix, and 
##      the data needed for Figures 1, 2, A3, A4, A5, and A6:

##      run the code ./master_tables.do
##
##      The file: replication_results.log is the log file with all the results.

##	To reproduce all the figures in the paper  
##      and appendix: run the code ./master_figures.do
